Recommended Step: Link to a Facebook Page

Even if you don't use Facebook, creating a "bridge" Page is often required for API access.

  1. Open Instagram App -> Profile -> Edit Profile.
  2. Under "Public Business Information", tap Page.
  3. Connect an existing Page or create a new one.
  4. Once linked, come back here and try again.
